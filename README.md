# California Housing — ML Models: From Scratch vs Scikit-learn

A comparative study implementing multiple machine learning algorithms **from scratch** and benchmarking them against their **scikit-learn** counterparts on the California Housing dataset.

## Overview
This project implements and compares the following models:
- Decision Tree
- Random Forest
- Linear Regression
- Logistic Regression
- KNN
- Naïve Bayes

Each algorithm is built manually (from first principles) and then compared against the equivalent scikit-learn implementation in terms of performance and accuracy.

## Tech Stack
- Python
- NumPy / Pandas
- Scikit-learn (for benchmarking)
- Jupyter Notebook

## Files
- `california_housing_ml_comparison.ipynb` — full implementation and analysis
- `California_Housing_ML_Report.pdf` / `.docx` — written report with findings

## How to Run
```bash
pip install numpy pandas scikit-learn matplotlib jupyter
jupyter notebook california_housing_ml_comparison.ipynb
```
